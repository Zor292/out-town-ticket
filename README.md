# 🎫 OUT TOWN - اوت تاون | Ticket Bot

بوت نظام التذاكر الرسمي لسيرفر **OUT TOWN - اوت تاون**

---

## 🚀 التثبيت

```bash
npm install
```

## ⚙️ الإعداد

انسخ ملف `.env.example` وأعد تسميته إلى `.env` ثم أدخل القيم:

```env
TOKEN=توكن_البوت
CLIENT_ID=معرف_البوت
GUILD_ID=معرف_السيرفر
```

## ▶️ التشغيل

```bash
npm start
```

## 📋 الأوامر

| الأمر | الوصف | الصلاحية |
|-------|-------|----------|
| `/panel` | إرسال لوحة التذاكر | مدير |
| `/close` | قفل التذكرة الحالية | رول الإدمن |
| `/add @مستخدم` | إضافة مستخدم للتذكرة | رول الإدمن |
| `/remove @مستخدم` | إزالة مستخدم من التذكرة | رول الإدمن |
| `/stats` | إحصائيات التذاكر | مدير |

## ⭐ نظام التقييم

- عند قفل أي تذكرة يُرسل تلقائياً:
  - **إمبد تقييم داخل التذكرة** — يُنشن فاتح التذكرة
  - **إمبد تقييم بالخاص (DM)** — لفاتح التذكرة
- لا يمكن التقييم أكثر من **مرة واحدة** لكل تذكرة
- يُرسل التقييم تلقائياً في قناة التقييمات: `1476912806057218048`
